# 🎉 ManxiAI 完整项目已创建完成！

## 项目概述

ManxiAI 是一个基于 Django + Vue 3 的企业级AI知识库系统，模拟MaxKB的核心功能，提供完整的前后端解决方案。

## 🚀 技术栈

### 后端技术栈
- **框架**: Django 4.2 + Django REST Framework
- **数据库**: PostgreSQL + pgvector
- **任务队列**: Celery + Redis
- **AI框架**: LangChain + OpenAI
- **文档处理**: PyPDF2, python-docx, openpyxl
- **向量化**: Sentence Transformers
- **部署**: Gunicorn + Docker

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **样式**: SCSS

## 🚀 快速启动

### 1. 后端启动
```bash
cd ManxiAI
pip install -r requirements.txt
python start.py migrate
python start.py createsuperuser
python start.py runserver
```

### 2. 前端启动
```bash
cd ManxiAI/frontend
npm install
npm run dev
```

### 3. 访问应用
- 后端API: http://localhost:8000
- 前端应用: http://localhost:3000
- API文档: http://localhost:8000/swagger/
